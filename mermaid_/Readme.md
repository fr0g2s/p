# Hello mermaid

### 주제
사용자로부터 도시 연결 정보가 적혀있는 파일를 입력받아 그래프로 보여준다.  
- 도로로 연결된 도시 쌍을 개행 기준으로 구분한다.  
- 도시와 도시는 ','를 기준으로 구분한다.

**example.txt**  
부산,서울  
부산,김해  
서울,평택




### 구성
|속성|의미|
|-----|--------|
|노드|도시|
|간선|연결된 도로|
|데이터 형식|각 도시들의 연결 정보가들어있는 2차원 배열|

### 도구
mermaid  
flask


### 주의
파일 경로 접근이 리눅스 기반임.
